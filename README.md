# Measure CPU power tool

Measure cpu package power consumption.

# Install

```
dotnet tool update DeviceTool.CpuPower -g
```

# Usage

Run as Administrator.

```
PS D:\> cpupower
AMD Ryzen 9 5900X
Avg: 61.22
Min: 56.49
Max: 72.67
```
